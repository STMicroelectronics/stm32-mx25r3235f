Purpose

The MX25R3235F Component Driver is a driver for the STM32Cube.

This software implements a Component Driver running on STM32 for Macronix MX25R3235F QSPI Flash memory.

It is built on top of STM32Cube software technology that ease portability across different STM32 micro-controllers
